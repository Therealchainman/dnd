# Dwarf Wizard Character Companion

This repository contains a ready-to-play, team-focused dwarf Wizard and a
Markdown memory system that can grow with the campaign.

The character is **therealchainman**, a level-1 dwarf Wizard with the Sage
background, built for the 2024 revised rules (5.5e) with expanded official
options. The name, story details, starting level, and point-buy scores remain
easy to change.

## Quick Start

- Open `character/character-sheet.md` for the numbers.
- Open `character/playbook.md` for what to do during play.
- Open `character/spellbook.md` for spell preparation and use.
- Open `character/roleplay-guide.md` for personality and story hooks.
- Open `memory/current-state.md` for the latest campaign snapshot.
- Tell the agent what happened after each game; it will create a session note and
  update the relevant memory files following `AGENTS.md`.

Useful prompts include:

- “We reached level 3. Walk me through my subclass and spell choices.”
- “Our party is a paladin, rogue, druid, and me. Tune my prepared spells.”
- “Here are my notes from session 4. Update memory but flag anything unclear.”
- “I just reached level 5. Walk me through the choices and update the sheet.”
- “We rolled initiative against three ogres. What is my best opening turn?”

## Build in One Sentence

Stay behind the front line, make dangerous spaces with control spells, protect
allies with reactions and abjuration, use rituals to conserve slots, and let the
other characters shine when your setup gives them an opening.

## Rules References

The build is based on the official D&D Beyond pages for the
[2024 dwarf and Sage background](https://www.dndbeyond.com/sources/dnd/br-2024/character-origins),
[2024 Wizard](https://www.dndbeyond.com/sources/dnd/br-2024/character-classes#Wizard),
and [2024 feats](https://www.dndbeyond.com/sources/dnd/br-2024/feats).
The DM's campaign rulings always take priority.
